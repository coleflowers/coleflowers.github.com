怎么完整下载带有引用其他项目的代码？

有些项目中用到其他开源项目时，可以不用整合整个项目的代码，只是引用。如果单一的git clone对于被引用的项目的代码是没办法下载的，这个时候可以加--recursive参数进行克隆。(关于如何引用其他项目，可以参考:1.http://git-scm.com/book/zh/v1/Git-工具-子模块;2.http://segmentfault.com/q/1010000000670427)

git clone --recursive https://github.com/rs/SDWebImage.git
这样可以下载链接的其他 git 项目
